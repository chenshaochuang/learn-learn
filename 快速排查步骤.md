# ⚡ 快速排查：部署后打不开

## 🔍 第一步：检查部署状态

1. **访问**：https://vercel.com/dashboard
2. **点击你的项目**
3. **查看最新部署的状态**：
   - ✅ **绿色 "Ready"** = 部署成功
   - ⏳ **黄色 "Building"** = 正在构建，等待完成
   - ❌ **红色 "Error"** = 部署失败

---

## 📊 如果显示 "Error"（部署失败）

1. **点击失败的部署记录**
2. **查看 "Build Logs"**（构建日志）
3. **告诉我错误信息**，我可以帮你解决

---

## ✅ 如果显示 "Ready"（部署成功）

### 尝试以下方法：

#### 方法 1：硬刷新浏览器
- **Windows**: `Ctrl + Shift + R`
- **Mac**: `Cmd + Shift + R`
- 或使用**无痕模式**访问

#### 方法 2：检查浏览器控制台
1. 打开部署的网站
2. 按 **F12** 打开开发者工具
3. 查看 **Console** 标签，看是否有错误
4. **告诉我错误信息**

#### 方法 3：检查项目设置
在 Vercel Dashboard 中：
1. 项目 → **Settings** → **General**
2. 检查：
   - **Framework Preset**: 应该是 `Vite`
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
3. 如果不对，修改后点击 **Save**

---

## 🔄 重新部署

如果以上都不行，尝试重新部署：

1. **在 Vercel Dashboard**：
   - 点击项目 → **Deployments**
   - 找到最新部署 → 点击 **"..."** → **"Redeploy"**

2. **或推送代码触发重新部署**：
   ```powershell
   git add .
   git commit -m "Fix deployment"
   git push
   ```
