## 🔍 如果打不开，按以下步骤排查

### 步骤 1：检查浏览器控制台

1. **打开部署的网站**（上面的任一地址）
2. **按 F12** 打开开发者工具
3. **查看 Console 标签**，看是否有红色错误
4. **查看 Network 标签**，检查资源是否加载成功

**常见错误**：
- `Failed to load resource` - 资源加载失败
- `404 Not Found` - 文件找不到
- `CORS error` - 跨域问题
- `Uncaught TypeError` - JavaScript 错误

---

### 步骤 2：尝试硬刷新

**Windows**：
- `Ctrl + Shift + R` 或 `Ctrl + F5`

**Mac**：
- `Cmd + Shift + R`

**或使用无痕模式**：
- `Ctrl + Shift + N`（Chrome）或 `Ctrl + Shift + P`（Firefox）

---

### 步骤 3：检查 Vercel 项目设置

1. **在 Vercel Dashboard** → 点击你的项目
2. **Settings** → **General**
3. **检查以下配置**：
   - ✅ **Framework Preset**: `Vite`
   - ✅ **Build Command**: `npm run build`
   - ✅ **Output Directory**: `dist`
   - ✅ **Install Command**: `npm install`

如果不对，修改后点击 **Save**，然后重新部署。

---

### 步骤 4：查看构建日志

1. **在 Vercel Dashboard** → 点击你的项目
2. **Deployments** → 点击最新的部署
3. **查看 "Build Logs"**
4. **检查是否有警告或错误**
